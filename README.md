# KeyGenerator
A Telegram bot where the admin generates one-time keys and users can check them for activation. Simple and convenient key management system. This is just a part of the code, useless on its own. To work, it needs to be integrated into your own bot.


Бот для Telegram, который позволяет администратору генерировать одноразовые ключи, а пользователям проверять их на активацию. Простая и удобная система ключей. Это всего лишь часть кода, сама по себе она бесполезна. Для работы её нужно встроить в вашего бота.
